# CV Builder Privacy Policy

Static privacy policy page for the **CV Builder** Android app (`com.cvbuilder.mobile`).

## Live page

After GitHub Pages is enabled for this repository, the policy will be available at:

https://dadrezaei.github.io/cv-builder-privacy-and-policy/

## GitHub Pages setup

1. Open the repository **Settings**.
2. Go to **Pages**.
3. Under **Build and deployment**, set **Source** to **Deploy from a branch**.
4. Choose the `main` branch and the `/ (root)` folder.
5. Save. GitHub will publish `index.html` from the repository root.

Use the live URL above in the Google Play Console **Privacy policy** field.
